Blog 1: What is "public static void main"?

   

As we have seen so far... every Java program starts with some form of:


public class " " {
     public static void main(String[] args) {


So what exactly does the 'public static void main' mean??

  • 'public':   "Access Modifier"  that allows this method to be called 'globally'.
  • 'static':   "Keyword"  that makes this method class related. It allows the JVM to be able to call it without creating an object.
  • 'void':   "Keyword"  that states that the method will not return anything... (return type void).
  • 'main':   "Method Name"  Main is the name of the method. This is what the JVM looks for first, and is the starting point of Java programs.
